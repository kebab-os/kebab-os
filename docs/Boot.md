## Boot Guide

In order to boot kebabOS, make sure that you are in the root directory (inside the `kebab-os` directory), then (on linux or other), run the following:

```sh
python main.py
```

---

If this fails to work, please [raise an issue](https://github.com/kebab-os/kebab-os/issues/new/choose), and we will help to sort it out.
